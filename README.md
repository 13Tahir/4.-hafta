# 🚀 Uzay Görev Merkezi (Void Task Hub)

Uzay temalı, modern, dinamik ve tam donanımlı bir görev yönetim (Task Manager) uygulaması. React ile geliştirilen estetik ön yüzü, Express.js ve MongoDB ile güçlendirilmiş sağlam bir backend yapısıyla birleştirir.

![GitHub repo size](https://img.shields.io/github/repo-size/kullaniciadi/repo-adi)
![GitHub license](https://img.shields.io/github/license/kullaniciadi/repo-adi)

## ✨ Özellikler

-   **🌌 Estetik Uzay Tasarımı:** Glassmorphism (Cam görünümü) ve hareketli yıldız arka planı.
-   **🛠️ Modüler Backend:** Express.js ile temiz API mimarisi ve Mongoose modelleri.
-   **💾 Kalıcı Veri:** MongoDB entegrasyonu sayesinde verileriniz asla kaybolmaz.
-   **🎭 Akıcı Animasyonlar:** `Framer Motion` ile kademeli yükleme ve yay efektleri.
-   **📋 Görev Akışı:**
    -   Görev oluşturma (Doğrulanmış inputlar).
    -   Görevi Üstlenme (Sorumlu atama sistemi).
    -   Durum takibi (Bekliyor, Aktif, Tamamlandı).
-   **🔍 Filtreleme ve Arama:** Görevler arasında anlık arama ve kategori bazlı filtreleme.
-   **📱 Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu.

## 🛠️ Teknoloji Yığını

-   **Frontend:** React, Tailwind CSS, Lucide Icons, Framer Motion, Vite.
-   **Backend:** Node.js, Express.js, Cors, Dotenv.
-   **Veritabanı:** MongoDB, Mongoose.
